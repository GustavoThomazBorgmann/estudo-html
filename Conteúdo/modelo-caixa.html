Modelo de caixas

Informações gerais

  • Aplicável ao HTML e CSS

Tipos de caixas

box-level

  • Sempre ocupa a largura total da tela;
  • Sempre se inicia em uma linha nova;
Tags
  • div
  • h1 - h6
  • p
  • main, headerm footer, aside;
  • video
  • article
  • form

inline-level

  • Não pula para a próxima linha;
  • Não quebra a linha;
  • Ocupa o tamanho que precisa;
tags
  • span
  • a
  • code
  • small
  • strong, b, i, em, u, sup, sub
  • buttom
  • select
